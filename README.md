# 🔍 SmoothySearch

A smooth and minimal app launcher for Linux, built with **Rust** and **Qt/QML** for Wayland.

## Features

- Fuzzy search
- Quick commands
- Built-in color presets + user themes
- Configure the launcher directly in the `Switch theme` menu
- Customizable window size and corner rounding

---

## Installation (Arch Linux)

[![Download](https://img.shields.io/badge/Download-latest%20release-blue?style=for-the-badge)](https://github.com/Wemite/smoothysearch/releases/latest)

1. Open a terminal in the folder where the package was downloaded.
2. Install the package: `sudo pacman -U smoothysearch-*.pkg.tar.zst`

---

## Run

**Launcher:** `smoothysearch`

To customize themes, type any query that returns no apps and choose `Switch theme`.
