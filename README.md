# 🔍 SmoothySearch

A smooth and minimal app launcher for Linux, built with **Rust** and **Qt/QML** for Wayland.

## ✨ Features

- Fuzzy search
- Quick commands
- Smooth animations
- 14 built-in color presets + example theme (Noir)
- Custom user themes and customizable window size & corner rounding

---

## 📦 Installation (Arch Linux)

[![Download](https://img.shields.io/badge/Download-latest%20release-blue?style=for-the-badge)](https://github.com/Wemite/smoothysearch/releases/latest) | ![Downloads](https://img.shields.io/github/downloads/Wemite/smoothysearch/total)

2. Open a terminal in the folder where the package was downloaded.
3. Install the package: `sudo pacman -U smoothysearch-*.pkg.tar.zst`

---

## 🚀 Run

**Launcher:** `smoothysearch`  
**Theme Editor:** `smoothysearch --themes`
